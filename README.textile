<i><strong>Teglon</strong> provides Clojure and RESTful APIs for managing, browsing, and searching Maven repositories.</i>

h2. Quick Start:

* @lein deps@
* @lein uberjar@
* @java -jar teglon-standalone.jar repository-directory@

Then visit 

http://localhost:8080
